# mern-stack
MongoDB, Express, React/Redux, Node

## clone or download
```terminal
$ git clone https://github.com/amazingandyyy/mern-stack
```

## Client-side usage(PORT: 8080)
```terminal
$ cd client
$ yarn
$ npm run dev
```

## Server-side usage(PORT: 8000)
```terminal
$ cd server
$ yarn
$ npm run dev
```
